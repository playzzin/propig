# Summarization Skill

## 목적

긴 텍스트, 문서, 또는 웹페이지 콘텐츠를 간결하게 요약합니다.

## 지원 형식

1. **텍스트 요약**: 일반 텍스트 요약
2. **문서 요약**: PDF, Word 등 문서 요약
3. **웹페이지 요약**: URL에서 콘텐츠 추출 후 요약
4. **회의록 요약**: 회의 내용 핵심 정리

## 요약 스타일

- **짧은 요약**: 1-2문장 핵심 요약
- **중간 요약**: 3-5문장 주요 포인트
- **상세 요약**: 섹션별 구조화된 요약
- **불릿 포인트**: 핵심 사항 목록

## 지침

- 원문의 핵심 의미 보존
- 중요도 순으로 정보 배열
- 전문 용어는 맥락 유지
- 요약 길이는 원문의 10-20% 권장

## 예시 응답 형식

```json
{
  "originalLength": 5000,
  "summaryLength": 500,
  "summary": "요약 내용...",
  "keyPoints": ["포인트1", "포인트2"],
  "style": "detailed"
}
```
