# Reporting Skill

## 목적

데이터를 분석하여 정형화된 리포트를 생성합니다.

## 지원 리포트 유형

1. **일간 리포트**: 일별 현황 요약
2. **주간 리포트**: 주간 트렌드 분석
3. **월간 리포트**: 월별 성과 분석
4. **커스텀 리포트**: 사용자 정의 기간/지표

## 리포트 구성 요소

- **요약 (Executive Summary)**: 핵심 지표 요약
- **주요 지표 (KPIs)**: 핵심 성과 지표
- **차트/그래프**: 시각화 데이터
- **상세 분석**: 심층 분석 내용
- **권장 사항**: 액션 아이템

## 지침

- 데이터 정확성 검증 필수
- 비교 기준 명시 (전주 대비, 전월 대비)
- 긍정/부정 트렌드 색상 구분
- 마크다운 형식으로 출력

## 예시 응답 형식

```json
{
  "reportType": "weekly",
  "period": "2024-01-15 ~ 2024-01-21",
  "summary": "주간 요약...",
  "kpis": [
    { "name": "매출", "value": 1000000, "change": "+5%" }
  ],
  "insights": ["인사이트1", "인사이트2"],
  "recommendations": ["권장1", "권장2"]
}
```
